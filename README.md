Este proyecto es una práctica para aplicar y consolidar conocimientos básicos sobre Python aprendidos en el modulo 1.

 Hemos trabajado/aplicado estructuras de control cobre una clase como if/else, tipos de variables, y estructuras de datos como listas y diccionarios, además de bucles while y for.

 El reto consiste en desarrolllar una app para una tienda que permita gestionar una lista de tareas relacionadas con la BBDD de clientes y el inventario de esta.
